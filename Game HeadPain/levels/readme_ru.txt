Здесь приведены обозначения символов для построения карт и их задних фонов с примерами.
По задумке все объекты на уровне используют светлые цвета, а фон тёмные (кроме ящика - он коричневый (тёмно-жёлтый)).
----------------------------------------------------
                      Объекты

----------------------------------------------------
                   Пример уровня

----------------------------------------------------
                     Задний фон
# или 0 = Чёрный
B = Тёмно-синий
G = Тёмно-зелёный
C = Голубой
R = Красный
M = Пурпурный
Y = Тёмно-жёлтый (+-коричневый)
W = Белый
S = Тёмно-серый
- = Серый
